## Revit Structure คืออะไร?

**Revit** คือซอฟต์แวร์ BIM (Building Information Modeling) ที่ใช้ในการสร้างโมเดลอาคาร 3 มิติ ซึ่งไม่ได้มีแค่รูปทรงทางเรขาคณิต แต่ยังเก็บ "ข้อมูล" ของอาคารไว้ด้วย (เช่น วัสดุ, ปริมาณ, ราคา, คุณสมบัติทางกายภาพ)

สำหรับ **Revit Structure** จะเน้นไปที่องค์ประกอบทางโครงสร้างโดยเฉพาะ เช่น เสา, คาน, พื้น, ฐานราก, และเหล็กเสริม (Rebar)

### ทำไมต้องใช้ Revit คู่กับ ETABS?

ใน Workflow การออกแบบตึกสูงยุคใหม่ เรามักจะใช้ 2 โปรแกรมนี้คู่กัน:

1.  **ETABS:** ใช้สำหรับ **"วิเคราะห์และออกแบบ"** (Analysis & Design) หาขนาดหน้าตัด และปริมาณเหล็กเสริมที่ต้องการ
2.  **Revit:** ใช้สำหรับ **"สร้างแบบก่อสร้าง"** (Documentation/Detailing) และถอดปริมาณงาน (BOQ) ตามผลลัพธ์ที่ได้จาก ETABS

## System Requirements (ความต้องการระบบ)

| รายการ      | ขั้นต่ำ               | แนะนำ                                    |
| ----------- | --------------------- | ---------------------------------------- |
| **OS**      | Windows 10 (64-bit)   | Windows 11 (64-bit)                      |
| **CPU**     | Multi-core 2.5 GHz    | 3+ GHz, 4+ cores                         |
| **RAM**     | 8 GB                  | 16 GB ขึ้นไป (โมเดลใหญ่ 30 ชั้น ใช้เยอะ) |
| **GPU**     | DirectX 11 compatible | NVIDIA Quadro / RTX Series               |
| **Disk**    | 30 GB SSD             | SSD 256 GB+                              |
| **Display** | 1280×1024             | 1920×1080 ขึ้นไป (จอ 2 จอยิ่งดี)         |

> [!WARNING]
> **RAM 8 GB ไม่พอสำหรับโมเดลใหญ่!** ถ้าคุณจะทำโมเดลคอนโด 30 ชั้นพร้อมเหล็กเสริม แนะนำ RAM **16 GB** เป็นอย่างน้อยครับ (32 GB ยิ่งดี)

## การติดตั้ง (Installation)

1.  ไปที่ **autodesk.com** -> ล็อกอินด้วย Autodesk Account
2.  ดาวน์โหลด **Revit** (เวอร์ชันล่าสุด เช่น Revit 2024 หรือ 2025)
    - _สำหรับนักศึกษา:_ สมัคร **Autodesk Education** ได้ฟรี 1 ปี
3.  รันไฟล์ `.exe` ที่ดาวน์โหลดมา -> กด **Install** -> เลือกภาษา `English`
4.  รอจนติดตั้งเสร็จ (ประมาณ 15-30 นาที) -> รีสตาร์ทเครื่อง
5.  เปิด Revit ครั้งแรก -> ล็อกอิน Autodesk Account -> เลือก License

เราจะจำลองสถานการณ์เหมือนทำงานจริง โดยใช้โปรเจกต์ **"คอนโดมิเนียมสูง 30 ชั้น"** (30-Story Condo) ที่เราได้ออกแบบไว้ใน ETABS มาขึ้นโมเดลใน Revit
