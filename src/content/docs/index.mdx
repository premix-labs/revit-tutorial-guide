## เส้นทางการเรียน

หนังสือเล่มนี้เรียงลำดับให้เรียนต่อเนื่องได้ตั้งแต่มือใหม่จนถึงระดับ production โดยใช้โปรเจกต์ตัวอย่างคอนโด 30 ชั้นเป็นแกนหลัก

### 1. พื้นฐานและการตั้งต้น

- บท 01-03: ทำความรู้จัก Revit, หน้าจอการทำงาน, การตั้งโปรเจกต์, units, grids, levels และการเตรียมไฟล์ให้พร้อมก่อนขึ้นโมเดล

### 2. ขึ้นโมเดลโครงสร้าง

- บท 04-07: ขึ้นเสา คาน พื้น ผนังรับแรง และบันได พร้อมเหล็กเสริมในส่วนที่จำเป็นสำหรับงานโครงสร้าง

### 3. คุมมาตรฐานและทำเอกสาร

- บท 14, 08, 09: วางมาตรฐานไฟล์, ตรวจ QA/QC, ทำ sheet และ schedule, แล้วค่อยเชื่อม workflow กับ ETABS หรือไฟล์ภายนอก

### 4. ระดับทีมและงานขั้นสูง

- บท 11-17: ทำงานร่วมกันด้วย worksharing, สร้าง family, ใช้ Dynamo, ทำ rebar documentation ระดับ production, ประสานงาน และส่งมอบงานจริง

### 5. ภาคผนวกอ้างอิง

- บท 10: ใช้เป็นคู่มือแก้ปัญหาเมื่อเจออาการที่พบบ่อยระหว่างทำงาน ไม่จำเป็นต้องอ่านต่อเนื่องตั้งแต่ต้น

## เหมาะกับใคร

- วิศวกรโครงสร้างหรือผู้เริ่มต้นที่ต้องการฝึก Revit Structure อย่างเป็นระบบ
- ผู้ที่มีผลออกแบบจาก ETABS แล้วต้องการนำมาทำโมเดลและเอกสารใน Revit
- ผู้ใช้งานที่ต้องการต่อยอดจากงานเดี่ยวไปสู่งานทีมและ workflow ระดับ production

## วิธีใช้หนังสือเล่มนี้

- ถ้าเริ่มใหม่ ให้เรียนเรียงตามลำดับบทใน sidebar
- ถ้ามีพื้นฐาน Revit แล้ว สามารถข้ามไปบท 04 เพื่อเริ่มขึ้นโมเดลได้
- ถ้าต้องการเน้นงานขั้นสูง ให้เรียนบท 11-17 หลังทำความเข้าใจบทพื้นฐาน, บทมาตรฐานงาน และบทเอกสารก่อน
